Topic 4: Links (Anchor Tags)

Basic Hyperlink

The a tag (anchor) creates a hyperlink. The href attribute specifies the destination URL.

code:
a href="https://www.example.com" Visit Example.com /a

Visit Example.com

Open Link in New Tab

Using target="_blank" opens the link in a new browser tab. rel="noopener noreferrer" improves security.

code:
a href="https://www.google.com" target="_blank" rel="noopener noreferrer" Open Google in New Tab /a

Open Google in New Tab

Relative Links (Internal Pages)

Relative links point to pages within the same website.

code:
a href="about.html" About Us /a
a href="contact.html" Contact /a
a href="images/photo.jpg" View Photo /a

About Us (example - create about.html to test)

Email Link

Use mailto: to create an email link that opens the user's email client.

code:
a href="mailto:someone@example.com" Send Email /a

Send Email

Phone Link

Use tel: to create a phone link (works on mobile devices).

code:
a href="tel:+1234567890" Call Us /a

Call Us

Anchor Links (Jump to Section on Same Page)

Create an anchor using id attribute, then link to it using #id.

code:
a href="#section-bottom" Jump to Bottom of Page /a

... (content) ...

h2 id="section-bottom" You have reached the bottom! /h2

Jump to Bottom of Page

Download Link

The download attribute tells the browser to download the file instead of navigating to it.

code:
a href="document.pdf" download Download PDF /a

Download PDF (example - need actual PDF file)

You have reached the bottom!
