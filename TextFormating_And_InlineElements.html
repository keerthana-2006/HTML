Topic 3: Text Formatting and Inline Elements

Bold and Strong Text

b tag makes text bold (visual only). strong tag makes text bold and indicates importance (semantic).

code:
p This is b bold /b text. This is strong strong /strong text./p

This is bold text. This is strong text.

Italic and Emphasized Text

i tag makes text italic. em tag makes text italic and adds emphasis.

code:
p This is i italic /i text. This is em emphasized /em text./p

This is italic text. This is emphasized text.

Underline, Mark, and Small

code:
p
    u underlined /u br
    mark highlighted /mark br
    small smaller text /small
/p

underlined
highlighted
smaller text

Deleted and Inserted Text

code:
p
    del This text is deleted /del br
    ins This text is inserted /ins
/p

This text is deleted
This text is inserted

Superscript and Subscript

code:
p
    E = mc sup 2 /sup (Superscript) br
    H sub 2 /sub O (Subscript)
/p

E = mc 2 (Superscript)
H 2 O (Subscript)

Preformatted Text (pre)

The pre tag preserves spaces and line breaks exactly as written.

code:
pre
    function greet() {
        console.log("Hello!");
        console.log("This spacing is preserved.");
    }
/pre
    
    function greet() {
        console.log("Hello!");
        console.log("This spacing is preserved.");
    }
