Topic 5: Images

Basic Image (img tag)

The img tag is used to embed images. src specifies the image path, alt provides alternative text for accessibility.

code:
img src="https://picsum.photos/id/1/300/200" alt="Mountain landscape" width="300" height="200"

Image with loading="lazy"

The loading="lazy" attribute delays loading images until they are near the viewport, improving page speed.

code:
img src="https://picsum.photos/id/15/300/200" alt="Forest path" width="300" height="200" loading="lazy"

Figure and Figcaption (Semantic Grouping)

figure groups the image with a caption. figcaption provides a visible caption.

code:
figure
    img src="https://picsum.photos/id/20/300/200" alt="Coffee cup"
    figcaption A delicious cup of coffee. /figcaption
/figure

Responsive Image with srcset

The srcset attribute provides different image versions for different screen sizes.

code:
img srcset="small.jpg 480w, large.jpg 1080w" 
     sizes="(max-width: 600px) 480px, 1080px" 
     src="fallback.jpg" 
     alt="Responsive demo"

Note: srcset requires actual image files to work properly.

Image as a Link

Wrap an img tag inside an a tag to make the image clickable.

code:
a href="https://www.example.com"
    img src="https://picsum.photos/id/26/200/150" alt="Clickable image"
/a

Image Attributes Summary

  • src - Image URL or path (required)
  • alt - Alternative text for screen readers (required for accessibility)
  • width - Image width in pixels
  • height - Image height in pixels
  • loading="lazy" - Defer loading off-screen images
  • title - Tooltip text on hover
