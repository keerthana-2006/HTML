Topic 1: HTML Basics and Document Structure

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using elements.

Basic Document Structure

Every HTML document follows this skeleton:

code:
!DOCTYPE html
html lang="en"
head
    meta charset="UTF-8"
    title Page Title
/head
body
    h1 My First Heading/h1
    p My first paragraph./p
/body
/html

Explanation of each part:

  • !DOCTYPE html - Declares this is an HTML5 document.
  • html - The root element of the page.
  • head - Contains meta information about the page.
  • title - Sets the title shown in browser tab.
  • body - Contains all visible content.
  • h1 to h6 - Heading tags (h1 is most important).
  • p - Paragraph tag for text.

Example: Complete HTML Document

code:
!DOCTYPE html
html
head
    title My First Webpage
/head
body
    h1 Welcome to HTML/h1
    p This is a paragraph of text.
    p This is another paragraph.
/body
/html

Comments in HTML

Comments are not displayed on the page. They help document your code.

!-- This is a comment and will not appear on the page --
