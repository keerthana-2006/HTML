Topic 7: HTML Tables

Basic Table Structure

A table is defined with table, tr (table row), th (table header), and td (table data).

code:
table border="1"
    tr
        th Name /th
        th Age /th
        th City /th
    /tr
    tr
        td John /td
        td 25 /td
        td New York /td
    /tr
    tr
        td Sarah /td
        td 30 /td
        td London /td
    /tr
/table
Name Age City
John 25 New York
Sarah 30 London

Table with thead, tbody, tfoot

Use thead for header rows, tbody for body content, tfoot for footer rows (semantic structure).

code:
table border="1"
    thead
        tr
            th Product /th
            th Price /th
        /tr
    /thead
    tbody
        tr
            td Laptop /td
            td $999 /td
        /tr
        tr
            td Mouse /td
            td $25 /td
        /tr
    /tbody
    tfoot
        tr
            td Total /td
            td $1024 /td
        /tr
    /tfoot
/table
Product Price
Laptop $999
Mouse $25
Total $1024

Merging Cells: colspan

colspan makes a cell span across multiple columns.

code:
table border="1"
    tr
        th Name /th
        th colspan="2" Contact Info /th
    /tr
    tr
        td John /td
        td john@email.com /td
        td 555-1234 /td
    /tr
/table
Name Contact Info
John john@email.com 555-1234

Merging Cells: rowspan

rowspan makes a cell span across multiple rows.

code:
table border="1"
    tr
        th Department /th
        th Employee /th
    /tr
    tr
        td rowspan="2" Engineering /td
        td Alice /td
    /tr
    tr
        td Bob /td
    /tr
/table
Department Employee
Engineering Alice
Bob

Table Caption

caption adds a title/description to the table.

code:
table border="1"
    caption Monthly Sales Report /caption
    tr
        th Month /th
        th Sales /th
    /tr
    tr
        td January /td
        td $5000 /td
    /tr
/table
Monthly Sales Report
Month Sales
January $5000

Complete Table Example

code:
table border="1"
    caption Employee Directory
    thead
        tr
            th ID /th
            th Name /th
            th Position /th
            th Salary /th
        /tr
    /thead
    tbody
        tr
            td 101 /td
            td Alice Johnson /td
            td Developer /td
            td $75,000 /td
        /tr
        tr
            td 102 /td
            td Bob Smith /td
            td Designer /td
            td $65,000 /td
        /tr
    /tbody
    tfoot
        tr
            td colspan="3" Average Salary /td
            td $70,000 /td
        /tr
    /tfoot
/table
Employee Directory
ID Name Position Salary
101 Alice Johnson Developer $75,000
102 Bob Smith Designer $65,000
Average Salary $70,000
