Topic 6: Lists

1. Unordered List (ul)

Unordered lists display items with bullet points. Use ul tag with li (list item) inside.

code:
ul
    li Apples /li
    li Bananas /li
    li Oranges /li
/ul
  • Apples
  • Bananas
  • Oranges

2. Ordered List (ol)

Ordered lists display items with numbers or letters. Use ol tag with li inside.

code:
ol
    li First step /li
    li Second step /li
    li Third step /li
/ol
  1. First step
  2. Second step
  3. Third step

Ordered List with Different Types

The type attribute changes the numbering style.

code:
ol type="A"
    li Item A /li
    li Item B /li
/ol

ol type="I"
    li Roman I /li
    li Roman II /li
/ol
  1. Item A
  2. Item B
  1. Roman I
  2. Roman II

3. Nested Lists

You can put lists inside other lists to create hierarchies.

code:
ul
    li Fruits
        ul
            li Mango /li
            li Kiwi /li
            li Apple /li
        /ul
    /li
    li Vegetables
        ul
            li Carrot /li
            li Broccoli /li
        /ul
    /li
/ul
  • Fruits
    • Mango
    • Kiwi
    • Apple
  • Vegetables
    • Carrot
    • Broccoli

4. Description List (dl)

Description lists are used for term-definition pairs. Use dl, dt (term), and dd (description).

code:
dl
    dt HTML /dt
    dd HyperText Markup Language - structure of web pages. /dd
    
    dt CSS /dt
    dd Cascading Style Sheets - styling and layout. /dd
    
    dt JavaScript /dt
    dd Programming language for interactivity. /dd
/dl
HTML
HyperText Markup Language - structure of web pages.
CSS
Cascading Style Sheets - styling and layout.
JavaScript
Programming language for interactivity.

List Attributes Quick Reference

  • ul - Unordered list (bullets)
  • ol - Ordered list (numbers/letters)
  • li - List item (used inside ul or ol)
  • dl - Description list
  • dt - Term in description list
  • dd - Description of the term
  • type - For ol: "1", "A", "a", "I", "i"
  • start - For ol: starting number (e.g., start="5")
  • reversed - For ol: counts downward
