Topic 2: Headings and Paragraphs

HTML Headings

Headings are defined with h1 to h6 tags. h1 defines the most important heading, h6 defines the least important.

Example of all heading levels:

code:
h1 This is Heading 1 /h1
h2 This is Heading 2 /h2
h3 This is Heading 3 /h3
h4 This is Heading 4 /h4
h5 This is Heading 5 /h5
h6 This is Heading 6 /h6

Actual Output:

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

HTML Paragraphs

The p tag defines a paragraph. Browsers automatically add a blank line before and after paragraphs.

Paragraph Example:

code:
p This is the first paragraph. It can contain multiple sentences. /p
p This is the second paragraph. There is a space between paragraphs. /p

Actual Output:

This is the first paragraph. It can contain multiple sentences.

This is the second paragraph. There is a space between paragraphs.

Line Breaks (br)

The br tag creates a line break without starting a new paragraph.

code:
p This is first line.br
This is second line.br
This is third line./p

This is first line.
This is second line.
This is third line.

Horizontal Rules (hr)

The hr tag creates a thematic break (horizontal line).

code:
p Above the line./p
hr
p Below the line./p

Above the line.

Below the line.
